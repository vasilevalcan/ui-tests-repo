Prerequisites to run frontend the tests
-Node Js
Download node-v16.19.1-x64.msi from https://nodejs.org/download/release/v16.19.1/
-Git
https://git-scm.com/download/win
Visual studio code
https://code.visualstudio.com/

Clone the REPO with the taskts
1. Create a folder on your local machine
2. Go to that folder
3. Open CMD and go to the new folder
4. In CMD run this command git clone https://github.com/vasilevalcan/ui-tests-repo.git

Install cypress
1.Open VS Code
2.Click Terminal->New Terminal
3.In terminal run command npm install cypress --save-dev --force
4.Run command npx cypress open
5.Select E2E testing option 
6 Select Electron and click Start E2E Testing in Electron

For a full tutorial on how to install cypress check the link below
https://www.way2automation.com/install-cypress-on-windows-os/
